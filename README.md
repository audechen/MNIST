### MNIST

**下载MNIST数据集，并打印一些基本信息：**

```
python download.py
```

**将MNIST数据集保存为图片**

```
python save_pic.py
```

**图像标签的独热表示**


```
python label.py
```

**Softmax 回归分类**

```
python softmax_regression.py
```

**两层卷积网络分类**
```
python convolutional.py
```
